 Title

Order can be completed with an empty shopping cart.

 Module

Checkout

 Severity

Medium

 Priority

High

 Environment

- Application: SauceDemo
- Browser: Safari (Version XX) *(və ya Chrome, hansında test etmisənsə)*

 Preconditions

- User is logged in.
- Shopping cart is empty.

 Steps to Reproduce

1. Log in with valid credentials.
2. Open the shopping cart.
3. Ensure the cart contains no products.
4. Click **Checkout**.
5. Enter valid First Name, Last Name, and Postal Code.
6. Click **Continue**.
7. Click **Finish**.

 Expected Result

The system should prevent users from completing the checkout process when the shopping cart is empty. An appropriate validation message should be displayed or the Checkout button should be disabled.

 Actual Result

The system allows the user to complete the checkout process successfully even though the shopping cart is empty.
