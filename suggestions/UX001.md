 Title

Clear validation state while editing input fields.

 Type

UI/UX Enhancement

 Description

After a failed login attempt, the username and password fields remain highlighted with a red validation icon while the user edits the input.

Removing the validation icon and error styling as soon as the user starts typing would provide more responsive feedback and improve the overall user experience.

 Expected Improvement

 Reduce visual distraction.
 Make the form feel more interactive.
 Allow users to focus on correcting their input.
