 SauceDemo Manual Testing Project

A comprehensive manual testing project created to practice Software Testing Life Cycle (STLC) concepts and build a professional Quality Assurance portfolio.

This repository contains the complete testing documentation for the SauceDemo web application, including test planning, scenario design, execution results, usability improvements, and supporting evidence.



 Project Information

| Item | Details |
|------|---------|
| **Application** | SauceDemo |
| **URL** | https://www.saucedemo.com |
| **Testing Type** | Manual Functional Testing |
| **Methodology** | STLC & Exploratory Testing |
| **Status** | In Progress |


 Project Objectives

- Practice manual software testing using a real web application.
- Learn how to create professional QA documentation.
- Design and execute functional test scenarios.
- Explore the application from an end-user perspective.
- Document observations and usability improvements.
- Build a portfolio that reflects industry-standard QA practices.
